PROJECTS

Here are some of the projects that I've worked on, including academic projects. Code is available on my GitHub unless it infringes on academic policies.

Lob Pong Game (April 2019, Java)

Created a simple lob pong game using java that moves the platform depending on the user's input in the GUI. Points were allocated based on a timer, which tracked the time the ball was in motion.

Features included:
  • .txt file that kept a mutable leader-board
  • Implementation of audio-related java libraries for startup sound
  • GUI for visual gameplay

Multi-Page Website (November 2019, HTML5, CSS, Javascript)

Designed and created a multi-page website using content from my classmates' and my own previous projects. Used a javascript plugin to incorporate a carousel on the home page.

Features included:
  • Home page with auto-sliding carousel
  • Implementation of Google Analytics for site analytics
  • Navigation bar that highlighted the current page
  • Auto-adjustments depending on the viewport size

Calculator (February 2020, Java)

Created a calculator that takes in equations from a text file and uses the shunting-yard algorithm to add things to a queue from a stack. The answers are returned in a separate file.

Features included:
  • Ability to look through a file and edit a file
  • Ability to follow PEMDAS
  • Handling of negative numbers
  • Graceful handling of invalid expresions
